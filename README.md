Una Landing Page responsiva y dinámica construida con **PHP puro** que consulta en tiempo real una API externa para mostrar cuántos días faltan para el estreno de la próxima película de Marvel, junto con los detalles de la siguiente producción.

---

##  Características

* **Datos en Tiempo Real:** Conexión mediante cURL a la API de lanzamientos de Marvel.
* **Diseño Minimalista y Responsivo:** Implementación de **Pico.css** para estilos semánticos rápidos y limpios.
* **Flexbox Layout:** Alineación inteligente y fluida controlada desde contenedores padre.
* **Modo Oscuro/Claro:** Switcher interactivo integrado con JavaScript para cambiar el tema del sitio.
* **Despliegue Contenerizado:** Configurado listo para la nube utilizando **Docker**.
